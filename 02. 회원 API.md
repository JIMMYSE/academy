# 회원 API

### 회원 구분
1. 총관리자: 백오피스 관리
   1. 회원 관리
      1. 회원 가입
      2. 회원 목록
      3. 회원 권한 부여: 교수자, 학습자 
   2. 클래스 관리 
      1. 클래스 목록
   3. 시스템 로그 확인
      1. 로그인 로그 (성공/실패 여부)
      2. 계정정보 변경 로그
      3. API 로그 (성공/실패 여부)
2. 교수자
   1. 클래스 관리
      1. 클래스 개설 / 종료
      2. 클래스 상세
3. 학습자

### 01. 회원 가입
- 백오피스에서 총관리자만 가입 가능
- 조건: 
    1. 4자리 ~ 20자
    2. 파일생성 불가 문자 제외 (\/:*?"<>|)

1. HTTP
~~~http request
POST https://0.0.0.0/v1/user/register
~~~

2. Request
- 교수자: 핸드폰 번호 필수
- 학습자: 학번 필수
    1. 교수자
    ~~~json
    {
       "user": {
          "id": "navy_01",
          "name": "신선향",
          "password": "test1234!@#$",
          "role": "10",
          "phone": "010-5555-5555",
          "identityNumber": null
       }
    }
    ~~~
    2. 학습자
    ~~~json
    {
       "user": {
          "id": "navy_01",
          "name": "신선향",
          "password": "test1234!@#$",
          "role": "20",
          "phone": null,
          "identityNumber": "210801"
       }
    }
    ~~~
    > role: 총관리자(00), 교수자(10), 학습자(20)

3. Response
~~~json
{
   "code": "S0001",
   "data": {
      "user": {
         "id": "navy_01",
         "name": "신선향",
         "role": "10"
      }
   }
}
~~~

### 04. 회원 정보 변경
- 총관리자
- 이름, 비밀번호, 전화번호, 학번, role 변경 가능
1. HTTP
~~~http request
PUT https://0.0.0.0/v1/user/[user ID]
~~~

2. Request
~~~json
{
   "user": {
      "phone": "010-3333-3333",
      "name": "홍길동"
   }
}
~~~

3. Response
~~~json
{
   "code": "S0001",
   "data": {
      "user": {
         "id": "navy_01",
         "name": "홍길동",
        "phone": "010-3333-3333",
         "role": "10"
      }
   }
}
~~~

### 05. 패스워드 변경
- 조건
    1. 숫자, 문자, 특수 혼합하여 9자 ~ 20자
    2. 아이디와 다르게
    3. 동일문자 3회 이상 반복 금지
    4. 오름차순, 내림차순 3회 이상 금지

1. HTTP
~~~http request
PUT https://0.0.0.0/v1/user/password
~~~

2. Request
~~~json
{
   "user": {
      "password": "test1234!@#$",
      "new_password": "1234!234test"
   }
}
~~~

3. Response
~~~json
{
   "code": "S0001",
   "data": {
      "user": {
         "id": "navy_01",
         "name": "신선향",
         "role": "10"
      }
   }
}
~~~
